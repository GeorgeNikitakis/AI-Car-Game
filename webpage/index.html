Some information about the project

AI playing a driving game

This project implements an AI trained using reinforcement learning and imitation learning to play a 2D car game. The object is to drive the car around the track and reach the goal without hitting the track boundaries or obstacles.

USER PLAYS

The user first plays the game and their actions are recorded. These actions are taken as starting point for decision making.

AI LEARNS

The saved information about user actions based are used by the AI to learn appropriate actions to take for different situations.

AI PLAYS

AI uses the learned behavior to drive around the track to reach the goal.

Game Features?

Below are a summary of features of the car driving game.

  • Tracks

    Three selectable tracks to choose from.

  • Visual Controls

    Visual indicator to provide feedback of inputs status.

  • Obstacles

    User placeable obtacles to change the track.

  • Game Status

    Text indicator for both game over and winning.
